# Add one or more vignettes to package

Add one or more vignettes to package

## Usage

``` r
add_vignettes(rmd_files, other_files = NULL)
```

## Arguments

- rmd_files:

  A character vector of .Rmd files, each corresponding to a vignette

- other_files:

  A character vector of any other files needed in the vignettes
  directory (.bib file, images, etc.)
