# Internal function for creating a .Rmd file from template

Internal function for creating a .Rmd file from template

## Usage

``` r
create_from_template(pkg_name, dir, template_name)
```

## Arguments

- pkg_name:

  Name of package to be created.

- dir:

  (Optional) Directory where .Rmd file should be created

- template_name:

  Name of template
