# Generate do-not-edit message to put at top of file

Generate do-not-edit message to put at top of file

## Usage

``` r
do_not_edit_message(rmd_file, type = c("R", "man", "c"))
```

## Arguments

- rmd_file:

  Name of the Rmd file to mention

- type:

  Whether this is a R/ file, man/ file, or a c file
